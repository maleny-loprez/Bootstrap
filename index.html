Programación WEB

Breadcrumb





Para indicar en que parte estamos situados, en el Proyecto de la materia de Programación Web de 7° Semestre.

Insignia

Para indicar la cantidad de elementos que usamos, en este caso fueron 7.

Modal

Esta ventana flotante, es un modal utilizado, así mismo también en el contenedor de los datos del estudiante.

Herramientas Responsivas

Es agregar clases para que el contenido se vaya adaptando según sea el tamaño del dispositivo donde se habra el sitio web.

Sidebar

Es el que contiene el menú lateral, en donde se muestra la ventana en donde actualmente estamos así como el dropdown de las herramientas utilizadas.

Botón Dropdown

Al oprimir dicho botón que se encuentra dentro del sidebar, despliega un menú con diferentes opciones que son enlaces.En este caso despliega las herramientas que utilizamos.

Tipografía de Textos

Es evidente que no todo el contenido de dicha página es igual y es ahí donde se aplica la tipografía.
